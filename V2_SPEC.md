## AI CAPABILITIES (System Architecture)

### AI Pipeline (Per Idea)

```
Input: User's idea description (1-3 sentences)
         ↓
Step 1: VALIDATE
  → Market size estimation
  → Competition analysis
  → Demand signals (Google Trends, social listening)
  → Monetization viability
  → Speed-to-launch score
  → Output: Idea Score Card (0-10 per dimension)
         ↓
Step 2: CONCEPT
  → Business name generation (3 options)
  → Value proposition
  → Target customer persona
  → Pricing strategy
  → Revenue model
  → Unique selling proposition
         ↓
Step 3: BRAND
  → Brand name (final selection)
  → Tagline / slogan
  → Color palette
  → Logo concept description
  → Brand voice guidelines
         ↓
Step 4: BUILD
  → Landing page HTML/CSS (complete, responsive)
  → Hero copy
  → Feature sections
  → Pricing display
  → CTA buttons
  → Email capture form
  → Stripe checkout integration
         ↓
Step 5: MARKET
  → 3 Facebook/Instagram ad variations
  → 5 social media posts (IG, X, TikTok captions)
  → 3-email welcome/nurture sequence
  → SEO meta tags
  → Launch announcement copy
         ↓
Step 6: LAUNCH
  → Deploy landing page to IdeaPark subdomain
  → Activate email capture
  → Connect Stripe (if paid product)
  → Generate shareable links
  → Activate analytics
```

## DATABASE SCHEMA (V2)

```sql
-- Users
users (id, email, name, avatar_url, plan, credits, stripe_customer_id, created_at)

-- Ideas (the core entity)
ideas (
  id, user_id, 
  -- Input
  raw_input, target_customer, price_range, differentiator,
  -- AI Validation
  score_market, score_monetization, score_speed, score_competition, score_overall,
  validation_report, -- JSON: full AI analysis
  -- AI Build Output
  business_name, tagline, value_proposition, target_persona,
  pricing_strategy, revenue_model, usp,
  brand_colors, -- JSON: primary, secondary, accent
  logo_description,
  landing_page_html, -- Full generated HTML
  ad_copy, -- JSON: array of ad variations
  social_posts, -- JSON: array of posts
  email_sequence, -- JSON: array of emails
  seo_meta, -- JSON: title, description, keywords
  -- Status
  status, -- submitted | validating | validated | building | ready | live | paused
  published_at, slug,
  -- Metrics
  visitors, signups, revenue,
  created_at, updated_at
)

-- Business visitors (analytics)
business_events (id, idea_id, event_type, visitor_ip, user_agent, referrer, created_at)

-- Email captures (per business)
email_captures (id, idea_id, email, source, created_at)

-- Marketplace (Phase 2)
builder_profiles (id, user_id, skills, hourly_rate, portfolio_url, bio, created_at)
collaborations (id, idea_id, builder_id, role, revenue_split, status, created_at)

-- Credit usage
credit_transactions (id, user_id, amount, action, idea_id, created_at)

-- Subscriptions
subscriptions (id, user_id, stripe_subscription_id, plan, status, created_at)
```
